# Portfolio – Saim Bahaeddine

Portfolio personnel moderne construit avec **Next.js**, **TypeScript** et **Tailwind CSS**.

## Démarrage

1. Installer les dépendances (depuis le dossier du projet) :

```bash
npm install
```

2. Lancer le serveur de développement :

```bash
npm run dev
```

3. Ouvrir `http://localhost:3000` dans le navigateur.

## Structure principale

- `app/page.tsx` : page d’accueil (sections Hero, Profil, Compétences, Projets, Parcours, Contact)
- `lib/profile.ts` : données structurées (profil, compétences, projets, etc.)
- `components/` : composants UI réutilisables (layout, cartes, badges…)

## Remplacer par vos données

Les textes et données sont centralisés dans `lib/profile.ts` pour être facilement mis à jour.
